# Play Colossal Cave Adventure with ChatGPT

## Usage

```
python3 adventure/main.py
```

## Setup

### Initial setup:

```
python -m venv venv
pip install
```

Add a `.env` file following `.env.example` with the correct values.

### Subsequent setup:

```
source venv/bin/activate
```


### Update requirements

```
pip freeze > requirements.txt
```
